# rover-lab
4WD outdoor skid-steering rover — build journal: mechanical design, electronics, firmware.

## Colour palette

Defined as CSS custom properties in the `<style>` block of each page.

| Token      | Hex       | Role                          |
|------------|-----------|-------------------------------|
| `--bg`     | `#0d0f14` | Page background               |
| `--bg2`    | `#13161e` | Card / section background     |
| `--bg3`    | `#1a1e28` | Elevated surface (hover, etc) |
| `--border` | `#232839` | Default border                |
| `--border2`| `#2e3448` | Stronger border               |
| `--text`   | `#e8eaf0` | Primary text                  |
| `--text2`  | `#9ba2bc` | Secondary / muted text        |
| `--text3`  | `#737a96` | Placeholder / disabled text   |
| `--accent` | `#4f8ef7` | Blue accent (links, headings) |
| `--accent2`| `#1a3a7a` | Blue tint (backgrounds)       |
| `--amber`  | `#f0a940` | Amber accent (warnings, tags) |
| `--amber2` | `#7a4d10` | Amber tint (backgrounds)      |
| `--green`  | `#3ecf7a` | Green accent (success, badges)|
| `--green2` | `#0d5c32` | Green tint (backgrounds)      |
